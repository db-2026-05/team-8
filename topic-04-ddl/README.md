## 📐 Tasks
For your chosen variant, complete the following:

| # | Task | Details |
|---|------|---------|
| **01** | 🧱 **Create full DDL schema** | Build all required database objects based on your finalized ER/conceptual design |
| **02** | 🔗 **Define relationships** | Add primary keys, foreign keys, and constraints needed for data integrity |
| **03** | **Add indexing strategy** | Create indexes for important queries and relationship columns |
| **04** | 🧹 **Keep script organized** | Use meaningful names, clean structure, and comments where needed |
| **05** | 🐘 **Ensure PostgreSQL compatibility** | Script must execute in PostgreSQL and produce a working schema |
| **06** | 📄 **Submit one SQL file** | Provide DDL as a single `.sql` file unless instructed otherwise |

> [!IMPORTANT]
> Your DDL output must fully match your approved ER diagram and conceptual schema.

> [!TIP]
> Separate script blocks by object type (tables, constraints, indexes) to make review and debugging easier.
